Sobre Nosotros

Nuestra Misión y Metodología

El objetivo principal de este trabajo ha sido diseñar una arquitectura de información eficaz que transforme una simple lista de canciones en una experiencia de exploración cultural. Para lograrlo, hemos aplicado una metodología centrada en el usuario:

  • Investigación de Usuarios: Realizamos análisis de Keywords y sesiones de Card Sorting para entender cómo los oyentes organizan mentalmente la música, priorizando el género y el formato físico.
  • Estructuración de Datos: Definimos taxonomías personalizadas (Formatos, Época, Procedencia) y describimos cada ítem utilizando metadatos estandarizados Dublin Core.
  • Implementación Técnica: La colección ha sido construida utilizando el generador de sitios estáticos Jekyll y el framework especializado CollectionBuilder, garantizando una web rápida, segura y sostenible.

Esperamos que esta colección sirva como ejemplo de cómo la correcta estructuración de contenidos puede enriquecer el descubrimiento musical.
